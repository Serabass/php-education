Устанавливаем Docker, чтобы не засорять систему.
https://www.docker.com/products/docker-desktop/

Клонируем проект:
```bash
git clone https://github.com/Serabass/php-education.git
```

Переходим в папку проекта:
```bash
cd php-education
```

Подниаем контейнеры:
```bash
up.bat - для Windows
up.sh - для Linux
```

http://localhost:8080/
